# sword - A word cutter in Go

`sword` will help you carve lines like `fatcat` into `fat cat`.

## Install

`go get github.com/kvap/sword`

## Docs

Detailed docs are available at <http://godoc.org/github.com/kvap/sword>.

## Tests

Just a `go test` will do.
